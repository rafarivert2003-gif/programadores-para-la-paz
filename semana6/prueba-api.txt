Se realizó una petición POST a la ruta /registro utilizando un objeto JSON con los campos "nombre" y "mensaje". El servidor recibió correctamente la información enviada y respondió con un objeto JSON que confirma que los datos fueron recibidos.

Respuesta obtenida:

{
  "estado": "Datos recibidos",
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

Esta prueba demuestra que la aplicación puede recibir datos enviados por el usuario mediante una petición POST y acceder a ellos utilizando req.body en Express.
